# Silsilah Keluarga - Family Tree

A web application for creating and managing family trees with interactive visualization.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** - Build tool
- **TailwindCSS** - Styling
- **D3.js** - Tree visualization
- **React Query** - Server state management
- **Zustand** - Client state management
- **React Hook Form** + **Zod** - Form handling & validation

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
